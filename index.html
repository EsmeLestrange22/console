Project on Console



VS Code is, in simple language, an interpretor. We use it to write codes in 3rd or 4th generation languages, and it converts those codes into the machine language, so that the computer can run them easily

A console is a screen that is part of every website, and displays errors, results, and lot more stuff as the user want to show on it. It appears when when use the keyboard shortcut- "Ctrl+Shift+I" or right click on the website screen and click on "Inspect". You can show something in it by using the following function:
"console.log("you can add variables, strings, etc here")"
